Pytania i odpowiedzi dotyczące stylu zewnętrznego CSS

  1. Składnia teoretyczna dla stylu zewnętrznego

    2. <link rel="stylesheet" type="text/css" href="styles.css">

  2. Gdzie wstawiany jest styl zewnętrzny, jaki znacznik jest za to odpowiedzialny oraz ile razy może on występować w dokumencie, jako podłączenie stylu CSS?

    3. Styl zewnętrzny jest wstawiany w sekcji <head> dokumentu HTML za pomocą znacznika <link>. Może on występować wielokrotnie w dokumencie HTML, co pozwala na podłączenie wielu arkuszy styli CSS.

  3. Jak jest najprostsza forma wstawiania stylu zewnętrznego?

    4. <link rel="stylesheet" href="styles.css">

  4. Ile arkuszy styli może być dołączanych do jednego dokumentu HTML oraz jaka zasad musi być zachowana przy większej ilości dołączanych styli?

    5. Do jednego dokumentu HTML może być dołączonych wiele arkuszy styli CSS. Zachować należy zasadę kaskadowości, która określa, że w przypadku konfliktu zostanie zastosowana deklaracja z bardziej specyficznego selektora lub z później dołączonego arkusza styli.

  5. Co się dzieje w przypadku konfliktów?

    6. W przypadku konfliktów, czyli gdy te same style są definiowane w różnych arkuszach styli podłączonych do tego samego dokumentu, zastosowana zostanie deklaracja z bardziej specyficznego selektora lub z później dołączonego arkusza styli.

  6. Jak są tworzone i edytowane pliki arkuszy z użyciem edytora tekstowego?

    7. Pliki arkuszy styli mogą być tworzone i edytowane za pomocą dowolnego edytora tekstowego, takiego jak Notepad++, Sublime Text, Visual Studio Code itp. Wystarczy utworzyć nowy plik, zapisać go z rozszerzeniem .css i rozpocząć edycję stylów CSS w tym pliku.

  7. Jak są tworzone i edytowane pliki arkuszy z użyciem specjalnego edytora CSS?

    8. Specjalne edytory CSS, znane również jako edytory CSS online, są narzędziami online do tworzenia i edycji stylów CSS w przeglądarce internetowej. Przykłady to CodePen, JSFiddle, CSSDesk itp.

  8. Jakie rozszerzenie posiadają pliki będący zewnętrznym arkuszem styli?

    9. Pliki będące zewnętrznymi arkuszami styli mają zazwyczaj rozszerzenie .css, na przykład styles.css.

  9. Jakie są zalety zewnętrznego arkusza styli?
    1. Oddzielenie treści od prezentacji.
    2. Możliwość wielokrotnego wykorzystania tego samego arkusza styli przez różne strony internetowe.
  10. Uwagi na temat pisania w CSS.

    11. Podczas pisania w CSS należy pamiętać o zasadach kaskadowości, dziedziczenia, specyficzności selektorów oraz optymalizacji kodu w celu poprawy wydajności i czytelności.
