Tabela: HTML vs CSS

HTML CSS
Określa hierarchię i związki między elementami Nie definiuje struktury dokumentu, koncentruje się na prezentacji
Stylowanie jest wbudowane w tagi Oddzielenie treści od prezentacji
Ograniczone możliwości stylizacji Bogate możliwości stylizacji.

Nowe możliwości CSS w porównaniu z HTML (wyliczenie numerowane)

  1. Elastyczne pozycjonowanie elementów na stronie
  2. Możliwość definiowania własnych stylów
  3. Zaawansowane Efekty Wizualne: CSS oferuje bogate możliwości stylizacji, takie jak gradienty, cienie, przejścia (transitions) i animacje, które dodają interaktywności i atrakcyjności stronom internetowym.
  4. Nowe Jednostki i Funkcje: CSS wprowadza różne nowe jednostki pomiaru i funkcje, takie jak jednostki viewport (vw, vh), które ułatwiają tworzenie bardziej dynamicznych i skalowalnych układów.

Wady CSS z wyliczeniem

  • Trudności w obsłudze niektórych elementów w starszych przeglądarkach
  • Brak modułowości: Trudno tworzyć odizolowane, modułowe style w czystym CSS.
  • Nadpisywanie stylów: Nadpisywanie styli przez różne selektory może być problematyczne i nieintuicyjne.
